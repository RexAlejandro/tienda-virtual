# Requerimientos
- xampp

---

# Importar 
1. Abrir xampp y habilitar Apache y MySQL.
2. Dar click en "admin" en MySQL.
3. En la pantalla que se despliega crear una nueva DB con el nombre "tienda_virtual" y importar la DB.
4. Guardar el codigo en la carpeta "htdocs".
5. Ingresar a la direccion "http://localhost/tienda-virtual/" desde el navegador.
